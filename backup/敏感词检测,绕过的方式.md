- 假设四个字
- 1和2互换，3和4互换
- 不影响阅读
```java
import com.github.houbb.sensitive.word.core.SensitiveWordHelper;

public class SensitiveWord {
    public static void main(String[] args) {
        System.out.println(SensitiveWordHelper.contains("斯大林"));
        System.out.println(SensitiveWordHelper.contains("大斯林"));
    }
}
```
